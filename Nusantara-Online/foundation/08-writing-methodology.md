# Writing Methodology — NUSANTARA ONLINE NOVEL

---

## Tone & Language

**Language:** Bahasa Indonesia sehari-hari, gaul tapi refined
**POV:** Third Person Limited (bergantian Arya/Bima/Chandra tiap 3 bab)
**Tense:** Present tense (lebih intens, kayak nonton anime)
**Vibe:** Novel webtoon Indonesia — action epic, emotional, relatable

---

## Per Bab Checklist

Setiap bab harus lolos 7 poin ini:

### 1. HOOK (3 kalimat pertama)
```
✅ WAJIB: Langsung menarik perhatian
❌ JANGAN: "Pagi itu, seperti biasa..."
✅ CONTOH: "Kode error merah memenuhi layar Arya. Sistem mendeteksi anomali. Dan anomali itu... adalah aku."
```

### 2. PROGRESI (50% bab = maju)
```
✅ WAJIB: Ada yang BERUBAH di setiap bab
❌ JANGAN: Bab filler yang gak majuin plot
✅ CONTOH: Bab baru = informasi baru / skill baru / hubungan baru
```

### 3. EMOTIONAL BEAT (minimal 1)
```
✅ WAJIB: Bikin pembaca marah / sedih / takjub / gemas
✅ TEKNIK:
  - Kejutan (plot twist kecil)
  - Haru (momen karakter)
  - Takjub (deskripsi epic)
  - Greget (adegan action)
```

### 4. CONFLICT (minimal 1 konflik)
```
✅ Jenis: Fisik / Emosional / Strategi / Sosial / Internal
✅ Aturan: Tiap 5 bab harus ada konflik besar
```

### 5. CLIFFHANGER (kalimat terakhir)
```
✅ Teknik Reveal / Interruption / Question / Betrayal / Deadline
❌ JANGAN: "Dan mereka pun tidur." atau bab berakhir antiklimaks
```

### 6. ANIME VISUALS (deskripsi)
```
✅ Gambarkan adegan kayak lagi nonton anime:
  - "Wajahnya close up, keringat menetes"
  - "Kamera wide shot: mereka berdua di tengah lautan"
  - "Slow motion: pedang dan petir bertemu"
```

### 7. DIALOG NATURAL
```
✅ Bahasa sehari-hari yang gak kaku
✅ Masing-masing karakter punya cara bicara:
  - Arya: singkat, langsung, kadang pake istilah teknis
  - Bima: banyak bercanda, persuasif, kadang lebay
  - Chandra: tenang, analitis, pake istilah probabilitas
✅ CONTOH (Bima): "Gue rasa... ini saatnya kita main keras. Lo siap?"
✅ CONTOH (Arya): "Probabilitas menang 34.7%. Itu sebelum kita mati."
```

## Teknik Menulis Adegan Action

### 1. Kalimat Pendek + Cepat
```
"BIMAAA!"
Terlambat. Pedang Guntur sudah turun.
Arya melompat. Kerisnya menyala. Sambaran petir.
"Jangan sentuh temanku."
```

### 2. Gunakan Panca Indera
```
Bukan: "Dia kena serangan."
Tapi: "Rasa panas menyambar lengan kirinya. Aroma hangus. Suara desisan."

Bukan: "Lautannya kencang."
Tapi: "Ombak setinggi 20 meter. Garam di bibir. Angin sayat di wajah."
```

### 3. Slow Motion di Momen Puncak
```
"Semuanya bergerak lambat. Chandra bisa melihat jalur anak panah — garis
putus-putus bercahaya di udara. Empat detik lagi kena. Tiga. Bima melompat.
Dua. Perisai... gak cukup. Satu. — ATAU INI YANG DIA RENCANAKAN?"
```

## Ending Chapter Formulas

| Tipe Ending | Efek | Contoh Last Line |
|-------------|------|-----------------|
| **Reveal** | Kaget | "Topeng itu jatuh. Dan di bawahnya... wajah yang sudah lama hilang." |
| **Interruption** | Penasaran | "Tapi sebelum dia selesai, langit di atas mereka... retak." |
| **Question** | Mikir | "'Gimana caranya? Kita baru level 30, bosnya level 200...'" |
| **Warning** | Deg-degan | "'Ada yang salah. Sistemku... dia mulai berbicara padaku.'" |
| **Promise** | Antusias | "Besok. Besok kita mulai perjalanan yang gak akan pernah mereka lupakan." |
| **Loss** | Sedih | "Skill terakhirnya menghilang. Bersama senyum terakhir Chandra." |
| **Victory** | Puas + penasaran | "Kita menang. Tapi di kejauhan... pintu baru terbuka." |

## Writing Workflow Per Hour

1. **Baca outline** (30 detik)
2. **Tulis bab** (40 menit)
   - 5 menit: tentukan hook
   - 25 menit: tulis konten ~2.500 kata
   - 5 menit: pastikan emotional beat
   - 5 menit: tulis cliffhanger
3. **Review & edit** (10 menit)
   - Cek hook
   - Cek cliffhanger
   - Cek dialog natural
   - Cek typo
4. **Save & format** (5 menit)
5. **Generate PDF** (5 menit)

---

## Quality Assurance

Sebelum push, cek:

- [ ] Hook di 3 kalimat pertama kuat?
- [ ] Ada progres di bab ini?
- [ ] Minimal 1 emotional beat?
- [ ] Cliffhanger bikin penasaran?
- [ ] Dialog natural gak kaku?
- [ ] Adegan action epic?
- [ ] Gak ada typo besar?
- [ ] Bab antara 2.500-3.000 kata?

## Catatan Penting

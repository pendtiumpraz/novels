# Skill System — NUSANTARA ONLINE

---

## Peringkat Skill

S分为:
- **E** — Basic (baru belajar)
- **D** — Intermediate (sudah paham)
- **C** — Advanced (master of basics)
- **B** — Expert (bisa kombinasi)
- **A** — Master (skill jadi instinct)
- **S** — Legendary (hanya sedikit pemilik)
- **SS** — Mythic (hampir dewa)
- **SSS** — Secret (9 skill saja)

## Level Up Skill

Setiap skill punya EXP bar. Naik level dengan:
- Menggunakan skill berulang
- Quest completion
- Sparring/PVP
- Training (meditasi, latihan)
- Skill Book

| Level | Nama | Buff |
|-------|------|------|
| Lv 1-10 | Pemula | Efek dasar |
| Lv 11-30 | Terampil | Efek 2x lipat, unlock sub-skill 1 |
| Lv 31-50 | Mahir | Efek 3x, unlock sub-skill 2, cooldown turun 20% |
| Lv 51-70 | Pakar | Efek 4x, unlock sub-skill 3, bisa combine |
| Lv 71-90 | Master | Efek 5x, cooldown turun 50%, bisa passive |
| Lv 91-99 | Grandmaster | Efek maksimal, unlock hidden effect |
| Lv 100+ | Transcendent | Efek melampaui batas skill |

## Combine Skill System

Saat 2+ skill mencapai Master, bisa dicombine:

| Kombinasi | Hasil | Contoh |
|-----------|-------|--------|
| Combat + Combat | Hybrid Attack | Pedang + Api → Pedang Berapi |
| Combat + Movement | Mobile Combat | Lompat + Tebas → Air Slash |
| Life + Combat | Combat Craft | Memasak + Api → Poison Chef |
| Social + Combat | Diplomatic Attack | Negosiasi + Elemen → Elemental Persuasion |
| Sense + Combat | Precision Attack | Hawk Eye + Panah → Snipe Lock |
| Communication + Combat | Coordination | Telepati + Combo → Sync Attack |
| Passive + Active | Auto-Skill | Regen + Pertahanan → Auto Barrier |

## Skill Trade & Transfer

### Market
Skill bisa diperjualbelikan di Market Hub:
- Harga tergantung rank + level
- Skill langka dilelang
- Pajak transaksi 5%

### Transfer
- Transfer = memberikan skill ke orang lain
- Yang memberi kehilangan skill permanen
- Skill dengan binding (Secret Skill) tidak bisa ditransfer
- Skill yang sudah dicombine: yang ditransfer hanya skill asli (combine bubar)

### Inheritance
- Mentor bisa mewariskan skill ke murid
- Mentor kehilangan 50% mastery skill itu
- Murid mendapat skill di level 1

## Skill Limit & Progression

Pemain bisa membawa maksimal:
- **Level 1-20:** 10 skill aktif
- **Level 21-50:** 20 skill aktif
- **Level 51-80:** 35 skill aktif
- **Level 81-99:** 50 skill aktif
- **Level 100+:** Unlimited (dengan syarat tertentu)

Sisa skill disimpan di Grimoire (buku skill) dan bisa ditukar kapan saja.
